# 推送到 GitHub

## 方法一：手动推送（推荐首次）

### 1. 初始化 Git 仓库

```bash
cd /Users/oubin/myData/哥飞课程/github-repo/gefei-seo-handbook
git init
git add .
git commit -m "Initial commit: 哥飞SEO全书 v1.0"
```

### 2. 在 GitHub 创建仓库

1. 打开 https://github.com/new
2. **Repository name** 填写：`gefei-seo-handbook`
3. 选择 **Public**（公开仓库）
4. **不要**勾选 "Add a README file"（本地已有）
5. 点击 **Create repository**

### 3. 关联并推送

```bash
git remote add origin https://github.com/32103210/gefei-seo-handbook.git
git branch -M main
git push -u origin main
```

### 4. 输入 GitHub 账号密码

- 用户名：`32103210`
- 密码：你的 GitHub Personal Access Token（不是登录密码）

> 💡 如果没有 Token：在 GitHub → Settings → Developer settings → Personal access tokens → Generate new token → 勾选 `repo` 权限

## 方法二：用 GitHub Desktop（更简单）

1. 下载 [GitHub Desktop](https://desktop.github.com/)
2. File → Add Local Repository → 选择 `/Users/oubin/myData/哥飞课程/github-repo/gefei-seo-handbook`
3. Publish repository 到 GitHub

## 设置 GitHub Pages（可选）

让书籍可以在网上浏览：

1. 进入仓库 → Settings → Pages
2. Source: **Deploy from a branch**
3. Branch: **main** , folder: **/ (root)**
4. 点击 Save

等待 2 分钟，打开：
```
https://32103210.github.io/gefei-seo-handbook/
```
