## 如何使用这些关键词

每一个词都可以作为建站方向。用法：

1. 选一个你感兴趣的词
2. 去 Semrush 查询该词的相关关键词
3. 用 KDRoi 公式筛选最值得做的词
4. 用搜索意图分析确定页面内容
5. 让 ChatGPT 生成网页
