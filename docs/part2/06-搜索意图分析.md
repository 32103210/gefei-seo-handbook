# 第6章：搜索意图分析

## 什么是搜索意图？

关键词选好后，必须分析搜索意图，才能知道如何制作页面。

同一个关键词，不同用户的意图可能完全不同：

- "iphone"：有人想了解产品，有人想去官网，有人想买，有人想比较
- "base64 decoder"：有人想了解原理，有人想直接用工具

**搞不清楚搜索意图，做出来的页面就是驴唇不对马嘴。**

## 四种搜索意图

| 类型 | 用户意图 | 页面应该 |
|------|---------|---------|
| **信息型（Informational）** | 用户想知道某事 | 提供知识性内容、教程 |
| **导航型（Navigational）** | 用户想去某网站 | 提供品牌/直接链接 |
| **商业型（Commercial）** | 用户想比较研究 | 提供评测、对比 |
| **交易型（Transactional）** | 用户想完成操作 | 提供工具、功能、购买 |

## 分析方法

**第一步：收集推荐词**

在 Google 搜索目标关键词，把光标放在不同位置，收集所有推荐词。

例如搜索 "phone number generator"，收集到：
```
phone number generator
phone number generator malaysia
phone number generator python
phone number generator github
phone number generator openai
phone number generator with sms
phone number generator for youtube
random phone number generator
us phone number generator
uk phone number generator
china phone number generator
california phone number generator
...
```

**第二步：让 GPT-4 分析**

把推荐词喂给 GPT-4，让它分析每种意图的用户真正想要什么。

提示词模板：
```
请详细解释一下 [关键词] 的搜索意图，用户到底想要什么服务
```

**第三步：基于分析设计页面**

GPT-4 分析 "phone number generator for YouTube" 的结果：
- **账户创建**：有些用户想创建多个 YouTube 账户
- **评论自动化**：有人想绕过手机验证来增加评论
- **测试开发**：开发人员需要虚拟号码用于测试

→ 基于这些分析，页面要明确：
- 工具的具体功能
- 合法使用场景
- 免责声明

## 实际演示

### 分析过程

在 Google 搜索 "phone number generator"，收集推荐词，去重后得到 27 个关键词。

然后让 GPT-4 分析 "phone number generator for YouTube"：

```
关键词 "phone number generator for YouTube" 的搜索意图可能涉及以下方面：

1. YouTube Account Creation
   有些用户可能希望创建多个 YouTube 账户，可能出于各种目的，
   如增加订阅、点赞或评论数量。他们可能正在寻找一种方式来
   生成多个电话号码，用于验证不同的 YouTube 账户。

2. Comment Automation
   有人可能寻求工具或服务，以帮助自动发布评论或互动在 YouTube 上。
   为了绕过 YouTube 的限制，他们可能需要多个电话号码来注册多个账户。

3. Testing and Development
   一些开发人员或测试人员可能需要电话号码生成器，
   用于创建虚拟测试账户，以便在开发或测试过程中模拟不同用户的行为。

需要注意的是，使用电话号码生成器来绕过验证或滥用平台的规则
可能会违反 YouTube 的政策，甚至可能触发法律问题。
```

### 基于分析设计页面

继续问 GPT-4：

```
对于 phone number generator for YouTube 这个词，
如果我要做一个网页来满足用户搜索需求，你有什么建议吗？
```

GPT-4 给出了详细建议：
- 明确目标和合规性
- 提供清晰的服务描述
- 用户教育和警告
- 免责声明
- 隐私保护
- 用户验证

## 总结

搜索意图分析是做好网站的关键一步。

不要想当然地认为用户想要什么，而是用数据和工具去验证用户的真实需求。

只有搞清楚搜索意图之后，才能知道怎么去满足用户的需求，才能知道我们的网站到底要怎么做。
