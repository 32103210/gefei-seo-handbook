# 第22章：广告收入估算公式

## 22.1 重新梳理四个核心指标

在进入公式之前，我们先把第 21 章提到的四个核心指标做一个系统性的梳理，确保你对每一个概念都能准确理解并运用。

### UV：独立访客

UV（Unique Visitor）是去重后的自然访问人数。在一天之内，同一个 IP 或同一个 Cookie 无论访问多少次，都只算 1 个 UV。

在 Google Analytics（现在叫 GA4）里，你可以直接看到日 UV 数据。在 Google Search Console 里，虽然不直接显示 UV，但通过"总点击数"和"平均排名"，你可以大致推算 UV 规模。

UV 是整个公式的基础，因为它代表了你覆盖的真实用户数量，是所有变现估算的起点。

### PV：页面浏览量

PV（Page View）是页面被加载的总次数。用户每访问一个页面，无论之前是否访问过，都记为 1 个 PV。

PV / UV 的比值，叫做平均访问深度（Average Depth）。这个比值越高，说明用户在你的网站上停留得越久、浏览的页面越多，对广告变现越有利。

以一个内容型网站为例，PV/UV 通常在 1.5 到 3 之间——用户在搜索结果页看到一个感兴趣的文章标题，点进去看完全文，可能就离开了。比值接近 1 说明用户"来了就走"，比值接近 3 说明用户会沿着内链深入浏览好几个页面。

工具类网站的 PV/UV 往往更高，因为用户会多次使用工具、查看不同的功能页面。

### ECPM：每千次广告展示收入

ECPM 的计算逻辑上文已经讲过，但这里要强调几个容易混淆的点。

第一，ECPM 不是固定的。它会随以下因素波动：

- **季节性**：每年 11 月到次年 1 月是广告竞价最激烈的时段（Holiday Season），广告主预算充足，ECPM 明显上升。电商旺季期间，教育、金融类广告主也会加大投放力度。
- **广告主预算**：重大节日（如中国的双十一、美国的黑五）前后，竞价广告价格上涨，ECPM 随之上扬。
- **填充率**：如果你的广告位没有被填满，展示次数减少，ECPM 会被人为拉低。填充率受网站流量质量、广告位位置、投放广告类型等多重因素影响。

第二，ECPM 的单位是美元，不是人民币。无论你的用户来自哪个国家，Adsense 后台显示的 ECPM 都是美元计价的。这意味着：一个 ECPM=4 的美国流量网站，月收入约等于一个 ECPM=0.4 的中国流量网站的 10 倍。

### 广告位数（Ad Slots per Page）

这是最后一个基础变量，指每个页面放置的广告位数量。上限是 3 个（Google Adsense 政策允许的最大值）。大多数情况下，你设置 3 个广告位，每个 PV 就等于 3 次广告展示。

但注意，这里说的是"广告位数量"，不是"广告数量"。一个广告位可以轮播展示不同的广告，但每次页面加载只算 1 次展示。广告位数量×广告展示次数 = 总广告展示次数。

## 22.2 核心公式推导

理解了上面四个变量之后，我们可以推导广告收入的核心公式。

### 从一个基本等式出发

我们先看一个最简单的情况：

- 网站某天获得 **1,000 个 UV**
- **PV/UV = 1**（即每个用户只访问 1 个页面）
- **ECPM = $1**（即每千次展示收入 1 美元）
- 页面有 **1 个广告位**

那么当天收入是多少？

1,000 个 UV × 1 PV/UV = 1,000 个 PV
1,000 个 PV × 1 个广告位 = 1,000 次广告展示
1,000 次广告展示 ÷ 1,000 = 1 个 ECPM 单位
1 × $1 ECPM = **$1**

所以：**1000 UV × 1 ECPM × 1 PV/UV × 1 广告位 = $1/天**

乘以 30 天，就是 **$30/月**。

这就是公式中数字"30"的来源：它代表的是"每个 ECPM 单位对应每千次展示的收入"，而一天有 24 小时，但我们用月来计算，所以是 30 天。

### 一般公式

将上述逻辑一般化，我们得到 Adsense 月收入的估算公式：

```
月收入(美元) ≈ 30 × UV基数 × ECPM × PV/UV × 广告位数
```

其中：

- **UV基数**：每日 UV 的千分之一（如果日 UV=10,000，则 UV基数=10）
- **ECPM**：每千次广告展示收入（美元）
- **PV/UV**：平均每用户访问页面数
- **广告位数**：每个页面放置的广告位数量（最大值 3）
- **30**：将日收入换算为月收入的系数（1 ECPM × 1,000 展示 = $1/天 → × 30 天）

### 公式的简化形式

如果你已经知道自己的日 UV，这个公式可以更直观地写成：

```
月收入 ≈ 日UV × PV/UV × ECPM × 广告位数 × 0.03
```

或者：

```
月收入 ≈ 日UV × PV/UV × 广告位数 × ECPM / 33.3
```

记住：日 UV 是最重要的变量，它决定了整个收入的天花板。ECPM 是乘数因子，SEO 做得好的网站，通过获取高质量流量，可以将 ECPM 放大 5-10 倍。

## 22.3 实战案例计算

现在，我们用一个具体的例子，把公式跑一遍。

### 案例背景

假设你有一个英文工具类网站，目标市场是美国，主要功能是在线 SEO 分析工具。

当前数据如下：

- 日 UV = 10,000
- PV/UV = 3（即每个用户平均访问 3 个页面）
- ECPM = $4（美国流量，这是相对保守的估计）
- 广告位数 = 3（每个页面 3 个广告位）

套入公式：

```
月收入 = 30 × 10 × 4 × 3 × 3
       = 30 × 360
       = $10,800
```

也就是说，这个网站每月 Adsense 收入约为 **10,800 美元**，折合人民币约 **78,000 元**（按 1:7.2 汇率计算）。

注意：这个估算是一个理论值，实际收入会受季节性波动、广告位填充率、广告展示的可见性（Viewability）等多重因素影响，真实收入通常在估算值的 70% 到 110% 之间波动。但作为量级估算，这个公式已经足够准确。

### 降低一个变量再看差距

为了让大家更直观地感受到各变量的影响，我们把 ECPM 从 4 降到 0.5（相当于中国流量），其他不变：

```
月收入 = 30 × 10 × 0.5 × 3 × 3
       = 30 × 45
       = $1,350
```

从 $10,800 跌到 $1,350，差距正好是 8 倍。

如果我们把日 UV 从 10,000 降到 1,000（也就是 SEO 做得一般的小站），其他条件不变：

```
月收入 = 30 × 1 × 4 × 3 × 3
       = $1,080
```

所以，**UV 是收入最核心的乘数因子**，ECPM 是最重要的质量因子。两者相乘，决定了你的最终收入。

## 22.4 英语工具站 vs 中文内容站对比

为了帮助大家建立更直观的收入预期，这里做一个跨市场对比。

### 英语工具站（美国流量）

| 指标 | 数值 |
|------|------|
| 日 UV | 5,000 |
| PV/UV | 2.5 |
| ECPM | $4.5 |
| 广告位数 | 3 |
| 月收入 | 30 × 5 × 4.5 × 2.5 × 3 = $5,062 ≈ ¥36,000 |

### 中文内容站（中国流量）

| 指标 | 数值 |
|------|------|
| 日 UV | 20,000 |
| PV/UV | 2.0 |
| ECPM | $0.6 |
| 广告位数 | 3 |
| 月收入 | 30 × 20 × 0.6 × 2.0 × 3 = $2,160 ≈ ¥15,000 |

中文站日 UV 是英语站的 4 倍，但因为 ECPM 只有英语站的 13%，最终月收入反而只有英语站的 42%。

**这个对比再一次验证了第 21 章的结论：面向英语国家市场的 SEO，是 Adsense 变现的最优路径。**

## 22.5 工具站 vs 内容站对比

除了市场差异，不同类型的网站在 Adsense 变现效率上也有明显差别。

### 工具类网站

工具站的用户行为特征是：高 PV/UV，深度使用。

以一个在线 SEO 工具网站为例，用户通常会：输入一个 URL 进行分析 → 查看详细报告 → 输入另一个关键词再次查询 → 查看对比报告。一个用户产生 5 到 10 个 PV 是常态，PV/UV 通常在 4 到 8 之间。

此外，工具站的用户往往是营销从业者、网站管理员，他们对广告的辨识度高，点击广告的概率相对较低，但广告主愿意为这类精准用户出更高价，因此 ECPM 通常较高。

综合来看，工具站的优势是：**PV/UV 高 → 广告展示次数多；用户精准 → ECPM 高。**两者叠加，是 Adsense 变现效率最高的网站类型。

### 内容类网站

内容站（博客、教程、资讯）的 PV/UV 通常较低（1.5 到 3），因为大多数用户看完一篇文章就离开了。

但内容站的优势在于：**容易获得长尾搜索流量**，关键词数量可以做得非常大。一个做得好的内容站，日 UV 达到几万甚至几十万并非难事，只是每个用户的 PV/UV 较低。

用公式来看，内容站的 UV 基数大，PV/UV 偏小，ECPM 取决于内容主题和目标市场。如果内容涉及金融、法律、健康等高价值领域，ECPM 可以和工具站比肩；如果内容是娱乐八卦、壁纸图集，ECPM 就会偏低。

## 22.6 如何用公式指导 SEO 决策

公式最重要的意义，不是让你算出一个精确的数字，而是帮助你做出正确的决策。

当你面对以下问题时，可以直接用公式来量化比较：

- **要不要换一个域名做英语市场？**——新站点 ECPM 的提升能否弥补前期 UV 损失，用公式跑一下就知道了。
- **要不要把一个 PV/UV 只有 1.2 的网站改成 SPA（单页应用）来增加内链深度？**——改版成本 vs PV/UV 提升带来的广告展示次数增加，公式可以帮你做权衡。
- **应该做 100 篇短内容还是 30 篇深度长文？**——短内容 UV 基数大但 PV/UV 低；长内容 UV 基数可能小但 PV/UV 高，用户停留时间长，ECPM 更高。公式可以把两种策略的收入预期量化对比。

理解公式，是 SEO 变现的第一步。用好公式，是 SEO 决策的核心能力。
