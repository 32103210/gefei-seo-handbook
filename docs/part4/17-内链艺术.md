# 第17章：内链建设的艺术

## 内链的三个层次

内链建设是 SEO 中最容易被忽视但最重要的环节之一。一个结构良好的内链系统，可以大幅提升网站的整体权重和用户体验。

### 第一层：首页 → 内页

首页是权重最高的页面，应该给每个重要内页一个链接。

**原则：用关键词作为链接锚文本**

```html
<!-- ✅ 好：锚文本包含关键词 -->
<nav>
    <a href="/us-phone-number-generator/" title="US Phone Number Generator">
        US Phone Number Generator
    </a>
    <a href="/china-phone-number-generator/" title="China Phone Number Generator">
        China Phone Number Generator
    </a>
    <a href="/india-phone-number-generator/" title="India Phone Number Generator">
        India Phone Number Generator
    </a>
</nav>
```

**为什么重要：**
- 首页权重最高，链接到内页能传递权重
- 关键词锚文本帮助搜索引擎理解内页主题
- 用户从首页可以直接进入各内页

### 第二层：内页 → 首页

内页是权重较低但内容最丰富的页面，应该把权重收回首页。

**原则：用核心关键词链接回首页**

```html
<!-- 在每个内页底部或侧栏添加 -->
<div class="back-to-home">
    <a href="/" title="Phone Number Generator">
        ← Back to Phone Number Generator
    </a>
</div>

<!-- 或者在内容中自然引用 -->
<p>
    If you need to generate phone numbers for other purposes,
    visit our <a href="/">main Phone Number Generator page</a>.
</p>
```

### 第三层：内页 → 相关内页

内容相近的内页之间，应该适当互链。

```html
<!-- 在"China Phone Number Generator"内页中 -->
<section>
    <h2>China Phone Number Format</h2>
    <p>...</p>
    <p>
        For generating numbers in other Asian countries,
        try our <a href="/japan-phone-number-generator/">Japan Phone Number Generator</a>
        or <a href="/korea-phone-number-generator/">Korea Phone Number Generator</a>.
    </p>
</section>
```

## 内链写法规范

**链接的三个关键属性：**

```html
<a href="目标URL" title="链接描述（关键词）">锚文本（关键词）</a>
```

| 属性 | 作用 |
|------|------|
| href | 链接目标 |
| title | 链接描述（给搜索引擎看的） |
| 锚文本 | 链接文字（关键词权重最高） |

## Nofollow 外链也有用

很多人以为 nofollow 外链是"浪费"，这是错误的。

Google 明确表示：**即使加了 nofollow，Google 可能仍然爬取这个链接**。

更重要的是，一个有大量外链引用的网站，本身就是权威性的体现。

## 首页展示新内容的艺术

把新内容在首页展示，是**屡试不爽的加快收录方法**。

```html
<section class="latest-updates">
    <h2>Latest Countries Added</h2>
    <ul>
        <li><a href="/bangladesh-phone-number-generator/">Bangladesh Phone Number Generator</a> — New!</li>
        <li><a href="/nigeria-phone-number-generator/">Nigeria Phone Number Generator</a> — New!</li>
    </ul>
</section>
```

**为什么有效：**
- 首页权重最高，新页面被链接后更容易被爬虫发现
- 每次更新首页 = 通知 Google 有新内容
- 新页面快速获得初始权重

## 面包屑导航

面包屑导航（Breadcrumb）是用户体验和 SEO 的双重利器：

```html
<nav aria-label="Breadcrumb">
    <ol>
        <li><a href="/">Home</a></li>
        <li><a href="/us-phone-number-generator/">US Phone Number Generator</a></li>
        <li aria-current="page">California</li>
    </ol>
</nav>
```

显示效果：`Home > US Phone Number Generator > California`

**SEO 好处：**
- 增加页面内链密度
- 帮助搜索引擎理解网站结构
- 生成更好的面包屑富文本搜索结果

## 内链常见错误

| 错误 | 后果 |
|------|------|
| 所有内页都链接回首页（无差异化） | 权重无法有效分配 |
| 内链锚文本都是相同文字 | 无法告诉搜索引擎各页面主题 |
| 内链放在页面最底部或隐藏 | 爬虫可能忽略 |
| 内链指向低质量内容 | 拉低整个网站的信任度 |

## 内链检查清单

每次更新网站后检查：
- [ ] 新页面是否在首页有入口？
- [ ] 新页面是否有链接回首页？
- [ ] 相关的内页之间是否互链？
- [ ] 所有链接是否正常工作（无死链）？
- [ ] 锚文本是否包含关键词（而非"点击这里"）？
