## 这是什么？ / What Is This?

**哥飞 SEO 全书** 是一本关于如何从零开始做出海 AI 工具站的实战手册，基于公众号「哥飞」120+ 篇原创文章整理，涵盖从关键词挖掘、网站制作、Google SEO 到 Adsense 变现的完整路径。

**The Gefei SEO Handbook** is a practical guide to building AI tools sites for global audiences, compiled from 120+ original articles on the WeChat public account "哥飞" (Gefei). It covers the complete path from keyword research, site building, Google SEO, to AdSense monetization.

**核心理念 / Core Philosophy：养网站防老（Nourish Your Website for Retirement）**
网站是资产，如同种树——你用心灌溉，它年年结果。SEO 带来的是可控、持久的搜索流量，不受算法波动和平台规则限制。

> A website is an asset, like planting a tree — with care, it bears fruit year after year. SEO delivers controlled, sustainable search traffic free from algorithm swings and platform restrictions.

## 快速开始 / Quick Start

### 第一步：选关键词 / Step 1: Choose a Keyword

从 **51 个财富密码关键词** 中选一个方向，筛选条件：

From the **51 money-making keywords**, pick a niche with these criteria:

```
搜索量 Search Volume ≥ 600/月
优化难度 KD（Keyword Difficulty）: 0–29
CPC（每次点击成本）> $0.1
```

用 KDRoi 公式排序：`KDRoi = (搜索量 × CPC) / KD`

Sort by KDRoi: `KDRoi = (Volume × CPC) / KD`

### 第二步：生成网站 / Step 2: Generate Your Site

参考 [第10章](docs/part3/10-ChatGPT生成网页.md)，把关键词告诉 ChatGPT：

Refer to [Chapter 10](docs/part3/10-ChatGPT生成网页.md), tell ChatGPT your keyword:

```
请帮我生成一个 SEO 友好的 [关键词] 工具网站，
包含：HTML + CSS + JavaScript，单页应用，
Title 和 Meta Description 已填写，
H1/H2 标签层级清晰，有面包屑导航，
最后加上 Google Analytics 和 AdSense 代码占位。
```

> 核心六字真言：**分门别类、罗列成语**
> Six-word mantra: **Categorize Everything, List Common Phrases**

## 核心公式 / Core Formulas

### KDRoi 公式（选词）KDRoi Formula (Keyword Selection)

```
KDRoi = (搜索量 × CPC) / 优化难度KD
KDRoi = (Search Volume × CPC) / Keyword Difficulty
```

数值越大，说明该关键词投资回报比越高。
Higher value = better ROI on SEO investment.

### Adsense 收入估算 / AdSense Revenue Estimation

```
月收入(美元) ≈ 30 × UV基数 × ECPM × PV/UV × 广告位数
Monthly Revenue ≈ 30 × Base_UV × ECPM × PV/UV × Ad_Slots
```

| 参数 | 说明 | Description |
|------|------|-------------|
| 30 | 常数：1000UV × $1 ECPM × 1页 × 1广告位 × 30天 | Constant: 1000UV × $1 ECPM × 1 page × 1 slot × 30 days |
| ECPM | 欧美 $3–6 / 中国 <$1 | EU & US: $3–6 / China: <$1 |
| PV/UV | 工具站通常 3–5 | Tools sites typically: 3–5 |
| 广告位 | 推荐 2–3 个 | Recommended: 2–3 slots |

**举例 / Example**：
日 UV=10,000 欧美工具站，ECPM=4，PV/UV=3，广告位=3
10K daily UV EU/US tools site, ECPM=4, PV/UV=3, 3 ad slots:

```
月收入 ≈ 30 × 10 × 4 × 3 × 3 = $10,800 ≈ ¥78,000
```

## 常见问题 / FAQ

**Q：我不会编程能做吗？/ Can I do this without coding?**
A：可以。ChatGPT 可以帮你生成全部代码，你需要的只是学习能力和执行力。Yes. ChatGPT generates all the code — you just need learning ability and execution.

**Q：需要多少钱？/ How much money do I need?**
A：域名约 $10/年，托管全部免费（Vercel + Cloudflare）。Domain ~$10/yr, hosting is free (Vercel + Cloudflare).

**Q：多久能看到流量和收入？/ How long until I see traffic and revenue?**
A：几天内收录，1–3 个月有流量，3–6 个月有稳定收入。Indexing in days, traffic in 1–3 months, stable revenue in 3–6 months.

**Q：SEO 要等多久才有效？/ How long does SEO take?**
A：SEO 是长期工程，时间是你的朋友。耐心做内容，搜索引擎不会亏待你。SEO is a long-term effort. Time is your ally — be patient with content, and search engines will reward you.

**Q：这本书适合谁？/ Who is this book for?**
A：独立开发者、不懂 SEO 的技术人员、想出海的个人站长、想建立副业的任何人。Indie hackers, developers unfamiliar with SEO, personal website owners going global, anyone building a side income.
