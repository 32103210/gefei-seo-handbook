## 快速开始

### 第一步：挖掘需求

从 **51 个财富密码关键词** 中选一个你感兴趣的词：

```
Translator, Generator, Calculator, Converter, Editor,
Maker, Downloader, Checker, Parser, Viewer ...
```

用 Semrush 查询该关键词，设置筛选条件：
- 搜索量 ≥ 600/月
- 优化难度 KD：0-29
- CPC > $0.1

用 KDRoi 公式排序，选出最值得做的词。

### 第二步：让 ChatGPT 生成网站

参考 [第10章：用ChatGPT生成网页](docs/part3/10-ChatGPT生成网页.md)，把关键词告诉 GPT-4，让它帮你生成 HTML/CSS 代码。

### 第三步：部署上线

参考 [第14章：注册域名和部署上线](docs/part3/14-域名部署上线.md)：
- 域名：Namecheap / Cloudflare Registrar
- DNS：Cloudflare（免费）
- 托管：Vercel（免费）

### 第四步：提交给 Google

注册 [Google Search Console](https://search.google.com/search-console)，提交网站，添加统计代码，等待收录。

## 核心公式速查

### KDRoi 公式（选词）
```
KDRoi = (搜索量 × CPC) / 优化难度KD
```

### Adsense 收入估算
```
月收入(美元) ≈ 30 × UV基数 × ECPM × PV/UV比 × 广告位数
```

**参数说明：**
- 基准值 30 = 1000 UV × 1 ECPM × 1 PV × 1广告位 = $1/天 × 30天
- ECPM：欧美 3-6 美元，国内 <1 美元
- PV/UV：工具站通常 3-5

**举例**：日 UV=10,000 的欧美查询工具站，ECPM=4，PV/UV=3，广告位=3

```
月收入 ≈ 30 × 10 × 4 × 3 × 3 = $10,800 ≈ ¥78,000
```

## 常见问题

**Q：我不会编程能做吗？**
A：可以。ChatGPT 可以帮你生成所有代码，你需要的是学习能力和执行力。

**Q：需要多少钱？**
A：域名约 $10/年，托管全部免费（Vercel + Cloudflare）。

**Q：多久能看到收入？**
A：最快几天可以拿到收录，1-3 个月开始有流量，3-6 个月开始有稳定收入。

**Q：SEO 要等多久？**
A：SEO 是长期工程。做好内容和外链，耐心等待，时间是的朋友。
